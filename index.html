HTML

  • The head element is a container for meta elements, meta elements holds information about the page that the browser uses
  • The head and meta elements arent seen by users
  • The body element shows visible content to users

CSS

  • Cascading Style Sheets
  • Three ways to style a webpage using css, inline css, internal and external css style sheets
  • internal style sheets contain css rules for the webpage in the head section of the HTML file, they only apply to that page
  • most common way to style is using an external css style sheet, separate file, ease of being a factor
  • external links the file using link element in the HTML's head
  • Selector defines the element or attributes to which the rules, or declarations, will apply
  • Declarations have two important components: the CSS property we want to apply and the value of the property
  • Padding adds space around the content inside the element, Margin property is space outside the element
  • **CSS Box Model**, HTML elements are essentially a box. Each box consists of margins outside the element, and the element's borders, padding, and content
  • * selector applies the rule to all visible elements
  • **DRY** Don't Repeat Yourself - two or more elements that will share the same stlyling can be grouped together. Combine two selectors into a single rule using a comma
  • **Separation of concerns**, commonly used term and refers to separating code into different files based on use. each section of code should have its own responsibility
